<h2>[ 문제풀이 ]</h2>  
이번에는 저번 문제와 반대로 NULL 값을 제외한 튜플을 뽑기 때문에 WHERE 절에 칼럼 IS NOT NULL을 걸어주면 된다.  

```java
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID ASC;
```
